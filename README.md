# 🔗 Linktree Pessoal

Projeto de uma página estilo Linktree desenvolvida com HTML5 e CSS3. O objetivo é reunir links importantes em um único local, permitindo que visitantes acessem rapidamente portfólio, GitHub, LinkedIn e Instagram através de uma interface simples e moderna.

---

## 🎯 Objetivo

Criar uma página de apresentação pessoal para centralizar redes sociais, projetos e informações profissionais em um único lugar.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Inter)

---

## ✨ Funcionalidades

- Foto de perfil personalizada
- Nome e descrição do usuário
- Botões de acesso rápido
- Efeito hover nos links
- Layout centralizado
- Design minimalista e responsivo

---
